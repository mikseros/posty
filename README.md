- Authentication
- Laravel Mix
- Controllers
- Routing
- Route model binding
- Database migrations
- Models
- Soft deleting
- Laravel string helpers
- Blade templates
- Blade components
- Blade directives
- Eloquent (Object Relational Map)
- Eloquent relationships types
- Model factories
- Authorization
- Sending email
- Validation
- Eager loading
- Pagination
- Redirecting
- CSRF protection
- Method spoofing
- Middleware
- Artisan console
- Dates and times
- Collections
